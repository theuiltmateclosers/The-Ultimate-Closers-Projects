💡 NOTES D'UTILISATION
═══════════════════════════════════════════════════════════

1. Téléchargez les fonts :
   - Playfair Display : https://fonts.google.com/specimen/Playfair+Display
   - Inter : https://fonts.google.com/specimen/Inter

2. Créez d'abord toutes les Color Styles (section COULEURS)

3. Puis créez tous les Text Styles (section TYPOGRAPHIE)

4. Utilisez Auto Layout dans Figma avec les espacements définis

5. Pour les boutons, utilisez :
   - Padding: 12px 32px (normal)
   - Padding: 16px 48px (large)
   - Border Radius: 9999px (full/pill shape)

6. Pour les cards :
   - Padding: 24px ou 32px
   - Border Radius: 12px ou 16px
   - Border: 1px solid #D6E5E3
